CSS - Box Sizing & Border Radius

Vamos ver nesta pagina como trabalhar com border-radius e box-sizing.

A pripriedade border-radius serve para criar um 'arredondamento' das bordas em um elemento, enquanto a propriedade box-sizing serve para controlar o tamanho do elemento. Vamos ver os exemplos a seguir

EXEMPLOS

border-radius

Titulo 2

Hic cillum ex sint, labore consequat qui exquisitaque. Sed ab culpa laboris a ita esse magna tamen voluptate e sunt tempor ne exercitation si est non multos lorem sint. Laborum labore cillum doctrina tamen quo nam labore consequat exercitation ubi an quae iis legam, culpa tempor te firmissimum.

Para colocarmos arredondamentos nas bordas, utilizamos a propriedade border-radius seguida do valor do arredondamento, este valor pode ser colocado em pixeis, quanto maior o valor, maior o arredondamento

Neste exemplo vamos utilizar border-radius de 5px.

Fazendo isso podemos ver como os cantos da nossa div ficaram mais arredondados.

box-sizing

Titulo 2

Et singulis aut consequat e quo esse doctrina imitarentur, o ne summis veniam nulla, varias iis an noster officia. Esse ex ex sunt iudicem, legam illustriora laborum veniam fabulas id fore ad hic magna expetendis o admodum noster singulis.

O box-sizing é muito utilizado, pois com ele podemos deixar o tamanho de um elemento, de acordo com o definido. No nosso caso o elemento tem uma altura e largura de 250px.

Se nos colocarmos um padding de 15px, teremos 30px a mais na altura e na largura, totalizando 280px.

Para forçarmos um elemento a utilizar o tamanho definido, independente do padding, utilizamos o valor border-box na nossa pripriedade

Se compararmos o titulo 1 com o titulo 2, os mesmo são identicos, porem no titulo 2 foi aplicado o box-sizing. Com isso podemos ver que o elemento passa a respeitar o tamanho definido por nós, mesmo com um padding de 15px aplicado, o tamanho continua o mesmo.
