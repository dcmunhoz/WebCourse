O que é?

O Ajax (Asynchronous Javascript and XML) em portugues significa 'Javascript e XML Assíncronos'. Ele nada mais é do que uma tecnica de programação que utiliza as tecnologias JavaScript e XML em sua construção.

Inicialmente o Ajax foi disponibilizado na versão 4.0 do Internet Explorer, porém com seu sucesso ele foi vastamente incorporado para outros navegadores
