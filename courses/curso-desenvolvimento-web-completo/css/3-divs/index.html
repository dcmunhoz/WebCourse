As divs por serem elementos block, sempre terão uma quebra de linha, como se existisse um BR antes e um BR depois da abertura e fechamento da TAG.
Vamos supor que você tenha um texto em uma div, e um pedaçõ desse texto você quer fazer uma formatação diferente, se você utiliza-se uma div entre os textos, iria ser feito uma quebrea de linha.
Para isso não ocorrer, utilizamos a tag SPAN.
