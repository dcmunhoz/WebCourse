CSS - Fontes e Cores

Podemos utilizar fontes diferentes em nossa pagina, essas fontes podem ser escolhides de dois tipos. Fontes presentes no proprio S.O. , e fontes externas.
Utilizando as fontes do proprio S.O., a pagina ira procurar no proprio computador a fonte da pagina.
Utilizando as fontes externas, é feito um download automatico das fontes para serem utilizadas na pagina.

Utilizando fontes do Sitema Operacional

Para utilizar fontes do S.O., vamos utilizar a seginte sintaxe:
font-family: [NomeFonte1], [NomeFonte2], [NomeFonte3];

Note que é possivel colocar mais de 1 nome de fonte, pois caso a pagina não encontre a primeira fonte no sistema operacional, ela ira procurar a segunda, e assim por diente.
Neste link podemos ver uma variedade de combinações seguras que podem ser utilizadas na sua pagina web. Combinações seguras são fontes que todos os sistemas operacionais possuem.

Utilizando fontes externas

Utilizar fontes externas é um passo mais complicado.

Primeiro de tudo, vamos precisar ter a fonte baixada e colocada em alguma pasta do nosso projeto.
Depois, no css devemos utilizar uma função chamada '@font-face{...}'.

Dentro do font-face vamos colocar os seguintes atributos:

font-family:[nome]
Nome da fonte externa que sera usada como referencia
src:[path]
Caminho de onde se encontra a fonte presente na pasta do nosso projeto.

Como exemplo vou utilizar uma fonte do site Google Fonts. Um repositórios de fontes gratis disponibilizado pela Google.

Para aplicar Cores nas fontes, precisamos apenas colocar o atributo 'color:[value]' no seletor, onde [value] é o nome da cor, podemos tambem utilizar codigos hexadecimais ou RGB. Existe tambem o RGBA que é utilizado para deixar uma fonte mais transparende ou solida.
