Bootstrap - Grids

Ao utilizar o Bootstrap, todo o layout passa a se comportar em um sistema de Grids.
A partir disso, vamos utilizar grids de até 12 colunas no total para montar nosso layout

Para definirmos nossas grids, no nosso elemento vamos definir uma classe chamada "col-n" onde N é o numero de grids que o elemento irá ocupar.
Podemos controlar quantas grids cada elemento vai possuir dependento do dispositivo/resolução que esta sendo visualizado.

  • .col-n Para dispositivos super pequenos , menores que 576px
  • .col-sm-n Para dispositivos pequenos, maiores ou iguais a 576px
  • .col-md-n Para dispositivos médios, maiores ou iguais a 768px
  • .col-lg-n Para dispositivos largos, maiores ou iguais a 992px
  • .col-xl-n Para dispositivos super largos, maiores ou iguais a 1200px

Vamos supor que no nosso layout teremos 2 elementosm, na visão desktop queremos que o elemento 1 ocupe 4 grids, então utilizamos a classe .col-lg-4 e no segundo elemento, utilizamos .col-lg-8, pois 12-4(quantidade utilizada no primeiro elemento) = 8
