Bootstrap - Containers

O primeiro elemento que devemos utilizar do bootstrap é o container , ele é como se fosse uma 'div' que comporta outros elementos, geralmente é utilizado na tag main.

Para uiliza-lo, basta nomear a class do elemento como .container.

Existem 2 tipos de containers, o .container e o .container-fluid.
A unica diferença é que o .container-fluid, ao dimensionar o navegador, o elemento responde de uma forma fluida, enquanto o .container não possui essa caracteristica.
Pode ser comparado com: width:300px e width: 90%
Utilizando % o elemento tem um dimensionamento mais fluido.

utiliza-se o fluid quando desejamos pegar 100% da nossa tela.
