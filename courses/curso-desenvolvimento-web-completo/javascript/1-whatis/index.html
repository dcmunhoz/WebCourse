O que é Javascript

  • Linguagem de programação de alto nível

    • Os scripts em Javascript possuem uma codificação que se assemelha mais a forma humana de conversar do que a linguagem de maquinas.

  • Client-Side (Executa pelo Cliente)

    • O javascript por ser client-side executa no lado do cliente, ou sejá, o proprio navegador executa.
      Tambem não é necessário instalar quaisquer ferramentas para executar o Javascript puro, o proprio navegador já vem com a linguagem imbutida.

  • Javascript não é Java

    • Muitas pessoas confundem achando que Javascript é uma distribuição Java para web, porem isso esta totalmente errado.
      Javascript é uma linguagem sem nenhum parentesco com o Java, ela possui muitos recursos, porem é muito mais compacta que o java.
      Java pode ser usado tanto cliente-side como server-side.

  • Possibilita tratar e aplicar dinamica ao elementos

    • Podemos manipular os elementos HTML5 pelo java script de modo dinamico, aplicando efeitos, utilizando para verificar se campos foram preenchidos etc...

  • Linguagem Interpretada

    • O Javascript não gera um executavel ou melhor dizendo, não é compilada.
      Os navegadores interpretam sua sintaxe em tempo real.
