CSS Intermediario - Elementos Flutuantes

Vamos aprender a como utilizar a propriedade Float em nossas div's(caixas).

Para utilizar o float, basta colocarmos da seguinte forma no seletor: 'float:[position];' onde [position] é a posição que ele ira flutuar, Exemplo:
float:left;
float:right;

Quando vamos criando mais elementos com o atributo float, um fluxo vai sendo criado, vamos perceber que quando não houver mais espaço na tela o proximo elemento do fluxo ira cair para a linha de baixo. Porem, existe um problema.
Apos todos os elementos floats terminarem, o fluxo ainda continuara na espera de um elemento float, sendo assim, o proximo elemento irá estar a direita do ultimo elemento, isso causa uma quebra de armonia no layout da pagina. Para isso, existe uma propriedade chamada CLEAR. Essa pripriedade serve para quebrar o fluxo deixado pelo float, o clear deve ser sempre utilizado no proximo elemento depois do float ou então pode ser uma div que não possua conteudo, porem a classe desta div precisa estar com a propriedado clear. Como utilizar:
clear: both;
O valor 'both' significa ambos os lados.
