Eventos Mouse Javascript

  • onclick

    Acionado no click do mouse sobre o elemento, necessário soltar o botão

  • ondblclick

    Acionado quando der dois cliques com o mouse

  • onmousedown

    Acionado exatamente ao precionar o mouse, não é necessário soltalo

  • onmouseover

    Acionado ao colocar o mouse em cima do elemento

  • onmouseout

    Acionado quando o mouse sair da area de um elemento
