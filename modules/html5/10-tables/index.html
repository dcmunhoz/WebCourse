Utilizando Tabelas

Para criar uma tabela, utiliza-se a tag 'table'.
Em conjunto com a tag table, é necessário a utilização de outras tags complementares, sendo elas:

  • 'caption' : Titulo que aparecera em cima da tabela
  • Grupos de uma tabela
    • 'thead' (Table Head): Cabeçalho da tabela
    • 'tbody' (Table Body): Corpo da tabela
    • 'tfoot' (Table Footer): Rodapé da tabela
  • Elementos de uma tabela
    • 'tr' (Table Row) : É utilizada para criar as linhas da tabela, dentro dela é possivel colocar outras tags, como 'th' e 'th'
    • 'th' (Table Head): Não confundir esta tag com a tag de grupo 'thead', a tag 'th' tem a função de destarcar o cabeçalho da COLUNA da tabela
    • 'td' (Table Data): É utilizada para informar o dado da presente na celula da tabela.

Para colocar bordas em uma tabela, é necessário utilizar o atributo 'border' com valores de 0 até n na tag 'table', onde quanto maior o numero, maior a grossura da borda.

Para aumentar a largura ou a altura de uma tabela, podemos utilizar o atributo presente em
quase todas as tags HTML, o 'width' e o 'height'.

Podemos tambem centralizar uma tabela utilizando o atributo 'align' e definindo seu valor para 'center'.

Porem sempre que for modifiar algo visual do código, é recomendavel a utilização de CSS
