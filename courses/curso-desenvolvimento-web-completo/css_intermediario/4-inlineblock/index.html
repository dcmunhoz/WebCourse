CSS Intermediário - Elementos Inline, Block, Inline-Block

Os elementos do HTML podem ser dividios em 2 dipos, elementos Inline e elementos Block.

Os elementos inline são elementos que quando colocados em sequencia, continuam na mesma linha, pois preenchem na tela apens o tamanho do proprio conteudo.

Já os elementos block, quando colocados em sequencia, quebram a linha para um abaixo do outro. Isso ocorre porque os elementos Block, independente do seu conteudo, sempre vão ocupar todo o espaço de vizulização, baseado no elemento pai.

Elementos Block

Temos os seguintes exemplos de elementos block:

  • <h1> - Cabeçalhos
  • <p> - Paragrafos
  • <div> - Divisão

Elementos Inline

Temos os seguintes exemplos de elementos inline:

  • <a> - Links
  • <span> - Span
  • <img> - Imagens
