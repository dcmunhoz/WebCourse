Requisições Sincronas e Assincronas

  • O que é uma requisição?
    • Como estamos trabalhando com desenvolvimento WEB, devemos lembrar que estamos trabalhando em um ambiente client-server, ou seja, a todo momento fazemos solicitações para o servidor onde o site esta hospedado, estas solicitações são as chamadas requisições.
  • O que é o método síncrono?
    • No método síncrono, quando enviamos uma solicitação, nosso browser fica aguardando até que o servidor retorne com a resposta dessas solicitação, ou sejá, cada requisição deve ser finalizada para que outra requisição sejá feita.
  • O que é o método assincrono?
    • No método assíncrono o browser consegue disparar diversas requisições sem a necessidade de esperar que 1 delas seja entregue para solicitarmos a outra. O método assíncrono trouxe uma grande vantagem em termos de usabilidade, permitindo que certos processos sejam feitos sem que a pagina de um Refresh.
