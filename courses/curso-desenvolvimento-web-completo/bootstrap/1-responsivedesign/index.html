O Bootstrap é um framework que nos permite trabalhar com design responsivo, o bootstrap utiliza Media Queries.

Um design responsivo, é um layout que faz o site ter a capacidade de se ajustar na tela do dispositivo em que esta sendo vizualidado, fazendo o site se ajustar

Se entrarmos no site do Media Queries podemo ver os exemplos que estão disponiveis na primeira pagina.
Quando a resolução do aparelho que estamos vendo é grande, o site se comporta de um jeito, ja quando é uma resolução menor, o site se comporta de outra forma.
