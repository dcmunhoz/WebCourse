HTML - Formulários

Para utilizar os formulários, devemos utilizar 2 tags importantes, a 'form' e a 'input'.
A tag 'form' serve para sinalizar o inicio do formulário, e dizer qual ação deve ser tomada com os dados presentes nela.

A tag 'input' serve para definirmos qual o tipo de campo vamos querer dentro do nosso forumlário, para isso, utilizamos o atributo 'type'
e nele podemos definir varios tipo de campos. Tambem é necessário informar o ID e o NAME das tags 'input', pois é com esses atributos que conseguiremos capturar os valores digitados futuramente.

Podemos tambem, atribuir uma tag 'label' antes do 'imput', como uma sinalização para o que deve ser preenchido, se é um campo de usuario, se é um campo de senha.

Abaixo estara uma lista de atributos que podem ser utilizados na tag 'input' e 'form'

Atributos
'action' : Define qual acão daquele form.
'type' : Define o tipo de campo que será utilizado
'id' : Identificador daquele campo. Utilziadao no CSS e JS.
'name' : Nome do campo.
'placeholder' : Como se fosse um label, porem fica dentro do campo.
'required' : Diz que o campo tem preenchimento obrigatório.
