JQuery - Eventos JQuery

  • O que são eventos?

    • Como ja vimos anteriormente, eventos são ações que podem ser capturadas conforme a utilização do usuário.
      Por exemplo, o clique do Mouse, uma teclas precionada.
      Podemos determinar uma certa ação a ser tomada com base nesse evento capturado.

  • Função ready() do JQuery

    • A função ready é altamente recomendavel que se use antes de iniciar uma codificação javascript, pois utilizando a função como na imagem abaixo, garantimos que todo codigo JQuery será executado somente quando todo o DOM for carregado.

      Isso nos da uma segurança a mais na hora de escrever o nosso codigo, pois temos a certeza que o elemento html já vai estar sendo interpretado pelo navegador antes de executarmos determinada função no mesmo.
