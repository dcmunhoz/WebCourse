Eventos Janela Javascript

  • onload

    Acionado assim que a pagina é totalmente carregada. O gatilho que despara o evento do onload deve ser informado no body ou em algum elemento que você queira que seja disparada a função assim que for carregado.

  • onresize

    Acionado assim que a pagina é redimensionada. Assim como o onload, gatilho deve ser colocado no body.
    Utilizar o console para entender melhor
