CSS Intermediario - Formatando Links

Os links tem sua edição no CSS diferente dos outros elementos, pois a tag do link possui estados diferentes. O que é isso?

Isso significa que podemos edidar os dipos de estados diferentes do link. Os estados são esses:

  • Normal - Estado em que o link se encontra assim que criado
  • Active - Quando precionamos o mouse sobre o link
  • Visited - Quando já visitamos aquele link, geralmente possui uma cor diferente dos outros links
  • Hover - Quando o mouse esta em cima do link

Estes são os estados mais comuns e mais utilizados de um link.
Vamos ao exemplo pratico:

Estado Normal

Para editarmos o estado normal, basta fazer como os outros elementos do html,
utilizando o seletor a{[proriedade]:[valor];...}

Google| Facebook

Estado Active

Para editarmos o estado active, apos colocarmos o seletor a{}, em frente o a, devemos colocar :active.
Exemplo: #active-link a:active{[propriedade]:[valor];...}

Google| Facebook

Estado Visited

Para formatarmos o estado Visited, utilizamos :visited.

Google| Facebook | Youtube

Podemos ver que quando um link já foi visitado, sua cor ficou com um tom alaranjado, diferente do link não visitado.

Estado Hover

Para editarmos o hover, utilizamos :hover

Google| Facebook

Ao passarmos com o mouse em cima do link, percebemos que sua cor é alterada.
