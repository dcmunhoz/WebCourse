Para quebrar linha no HTML, não basta simplesmente precionar ENTER Como agora, se inspecionado o codigo fonte, vera que existe uma quebra de linha, porem o navegador não reconhece Assim como varios espaços que poderão ser vistos ao inspecionar a pagina.

Para quebrar a linha, é necessário utilizar a tag BR.
Ao utiliza-lá podemos ver que o texto não continuou porem foi para a linha de baixo como esperado.
