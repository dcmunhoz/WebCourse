CSS - Seletores

A utilização de seletores no CSS é muito importante, pois com ela podemos dar uma 'identidade' a um determinado elemento.
Isso tambem nos permite utilizar arquivos externos de css, o que é a forma mais pratica e eficiente de utiliza-lo.
Existem dois tipos de seletores, os IDS e as CLASSES. IDS são elementos UNICOS, ele não pode ser repetido em outra tag, por outro lado, o seletor CLASS pode ser usado em mais de um elemento ao mesmo tempo, fazendo com quem sua edição sejá realizada em todos os elementos que possuem essa class.
Para utilizar os seletores, é necessário adicionar um atributo ' id="[nome]" ' ou ' class="[nome]" ' no atributo desejado, onde [nome] é o identificador daquele elemento.

Para referenciar um seletor no css, é necessário utilizar as formas de estilo internal ou external. Para utilização da forma interna, dentro da tag HEAD é necessário abrir uma nova tag STYLE com o atributo 'type=text/css'
para dizer que aquele estilo é um estilo css.

Neste formato, não necessáriamente deve-se utilizar somente seletores id e class, você tambem pode usar a tag como seletor, apenas informando o nome dela, porem, ao utilizar o nome da tag, caso tenha mais que 1 no arquivo html, todas elas serão aplicadas o efeito.

Para selecionar um id no css, devemos utilizar #[nome], onde nome é o identificador colocado no atributo. Para as classes deve-se utilizar .[nome].
