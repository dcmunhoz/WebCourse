CSS3 - Normalize

Antes de vermos a fundo o css3, vamos entender uma coisa.

O css pode ter alguns elementos renderizados de formas diferentes de acordo com o navegador utilizado. Por exemplo, a tag H1 no Chrome possui uma margem superior de 5px, se a mesma pagina for aberta no Firefox, esta margem pode ser de 3px por exemplo.

Isso faz com que nosso Layout tenha aparencias diferentes de acordo com o navegador utilizado. Como não queremos que isso aconteça, existe um arquivo css chamado Normalize que serve para "igualar" os estilos padrões para todos os navegadores.

Para utilizar o normalize, precisamos somente pesquisar no google 'normalize.css', e abrir a pagina do repositorio no github. Copiamos o código e criamos uma arquivo no nosso projeto, colamos o codigo e em seguida criamos um link com referencia para esse arquivo.

Pronto, o normalize esta instalado :D
