Array Javascript

  • O que é um array?

    • Um array é uma variavel que ocupa mais de 1 espaço na memória.
      Ele funciona como uma lista, por exemplo, podemos definir um array de lista de frutas e dizer que na posição 2 daquele array, temos uma laranja.

      Utilizamos array quando temos que definir uma lista de mesmo objeto, exemplos, uma lista de nomes de alunos.
      Para não criarmos 1 variavel para cada aluno, criamos 1 unico array.
