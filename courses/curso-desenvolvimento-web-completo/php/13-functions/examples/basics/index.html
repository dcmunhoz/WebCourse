Funções Basicas

Vamos agora aprender algumas funções basicas do PHP
  • isset()
    • O Isset verifica se uma variavel foi iniciada, os valores retornados são true ou false
  • empty()
    • O Empty é uma função que verifica se a variavel esta ou não vazia
  • is_numeric()
    • A função is_numeric verifica se o valor presente na variavel é um valor numerico ou não.
