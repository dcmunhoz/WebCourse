Funções php

  • is_array()
    • Esta função verifica se determinada variavel é um array, caso sim retorna TRUE senão FALSE
  • in_array()
    • Verifica se determinado conteudo esta presendo no array.
  • array_keys()
    • Retorna todas as chaves de uma array.
  • sort()
    • Ordena um array por ordem alfabetica
  • asort()
    • Ordena um array por odem alfabetica mantendo os indices em seus valores
  • count()
    • Conta quandos elementos estão presentes dentro de um array
  • array_merge()
    • Funde um ou mais array
  • explode()
    • Divide uma string baseado em um delimitador, transformando a nova variavel em um array
  • implode()
    • Junta os elementos de um array em uma string.
