HTML5 - Nova tags INPUT

Foram adicionadas novos types de INPUT no HTML5, como por exemplo:

  • email: tipo proprio para colocar endereços e-mail
  • search: tipo para formulários de pesquisa
  • url: Tipo para inserir URL's
  • tel: Tipo para inserir TELEFONE
  • number: tipo para informar numeros

Entre outros que vamos verificar no exemplo a seguir:

Search

Como descrito acima, search é um novo tipo de input do html5. Sua utilização é em formulários de pesquisa.

E-mail

O tipo E-mail, pelo próprio nome já sabemos do que se trata, serve para informarmos um endereço de email.

URL

O tipo url serve para informar um endereço web, neste endereço é necessário conter http://www.[site].com para ser uma url valida.

TEL

O tipo TEL é usado quando queremos informar algum telefone. Celulares com IOS e Android já reconhecem este tipo de input em seus navegadores, com isso automaticamente conseguem mudar o teclado para um tipo somente numerico

Number

O tipo number, serve para informarmos valores numericos, podemos notar que este tipo é renderizado diferente pelos navegadores. O tipo number possui 2 botões que servem para aumentar ou diminuir o numero selecionado.

Podemos tambem definir um valor minimo e um maximo utilizando as propriedades que podem ser vistar olhando o código fonte.

Date

Este novo tipe nos permite selecionar uma data. Ao clicar no campo um calendario é aberto para podermos selecionar a data desejáda.
