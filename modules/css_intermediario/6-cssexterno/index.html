CSS Intermediario - CSS Externo

Para aplicarmos um css externo, dentro do nosso projeto, vamos criar uma pasta chamada "css" e dentro dela vamos dar o nome de style.css.

Dentro desse arquivo, vamos colocar todo nosso codigo CSS.

Feito isso, na nossa pagina html que queremos utilizar aquele arquivo CSS, entre as tags HEAD, vamos criar um link para aquele arquivo da seguinte forma:

<link rel="stylesheet" type="text/css" href="css/style.css" />

Nos devemos abrir a tag link, e refinir o rel como "stylesheet", rel vem de relationship, traduzindo, relação. Em seguida devemos definir o type da relação, no nosso caso será "text/css". E por final como já aprendemo, temos que definr o href que é o link de referencia do nosso arquivo html. Como nosso arquivo é interno do projeto, vamos definir como "css/style.css", se fosse para um arquivo externo da Web, poderiamos colocar o link para tal.
