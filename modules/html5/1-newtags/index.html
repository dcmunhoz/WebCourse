HTML5 - Aprendendo as novas tags

No html5 foram inseridas novas tags. Essas tags dão mais sentido ao código e faz com que o navegador entenda melhor sua utilização.

Um exemplo, é quando criamos a div do corpo da nossa pagina, geralmente representada pelo id "MAIN". No html5 a tag main já esta presente, então em vez de utilizarmos uma div com id, podemos simplesmente usar a nova tag MAIN. Isso da mais sentido ao codigo e faz com que o navegador interprete melhor. As ferramentas de Search Engine tambem já estão configuradas para reconheçer essas tags, isso faz com que sistemas como Google e Bing encontrem facilmente seu site (Mais explicações curso de SEO).

Exemplos:

Site Exemplo

Main

A tag MAIN é o corpo da pagina

Header

A tag HEADER são os cabeçalhos da pagina. Toda seção que possuir um cabeçalho, podera ser utilizada a tag HEADER. O logo do site geralmente se encontra dentro desta tag

Nav

A tag NAV é qualquer elemento de navegação. Sejá uma barra superior ou lateral, geralmente é onde ficam os menus dos sites.

Article

A tag Article é onde se encontra o conteudo principal do site, uma noticia ou uma postagem de um blog.

Section

A tag Section é uma tag bem genérica, ela serve para separar o site em varias seções. A Section é como se fosse a propria tag Div, tem a sua utilização parecida, alem disso ela facilita os indexadores a lerem melhor o site, tambem ajuda nos leitores automaticos, para lerem os conteudo na ondem correta.

Time

A tag TIME serve para informar que o que esta dentro dela se trada de uma data ou horário.

Podemos tambem utilizar um atributo nesta tag, o atributo "datetime" que serve para relacionar a data do atributo com a presente entre as tags, facilitando tambem os indexadores a lerem melhor o site.

Asside

A tag Asside pode ser utilada sempre que tem um elemento lateral no site, pode ser um banner, um menu lateral (Tag NAV deve ficar dentro de ASSIDE), ou qualquer outra coisa que ficara na lateral do site.

Footer

A tag FOOTER serve para informar o rodapé do site, geralmente onde esta localizado o ©Copyright
