CSS3 - Animações

Vamos trabalhar com animações CSS3. Antes fazermos animações, precisavamos utilizar Javascrip ou Jquery , porem com o CSS3 podemos trabalhar com animações sem precisar desta outra linguagem.

Para fazermos uma animação, precisar abrir um parametro no CSS como se fossemos modificar um seletor, utilizando @keyframe [nome-animação]{}.
Dentro deste keyframe, vamos colocar qual é a animação que queremos, para isso devemos especificar em qual TEMPO sera feito essa animação. Podemos utilizar os seletores de animação como 0%{} até 100%{} , onde 0% é o inicio da animação e 100% é o final da animação.
E dentro desses seletores vamos colocar qual a animação. Por exemplo, movimentar o elemento 100px para a direita.
No 0%{} que é o inicio, colocamos da seguinte forma: 0%{left:0px;}
E no 100%{} que é o final, colocamos da seguinte forma: 100%{left:100px;}

Apos criarmos a animação, devemos aplicar esta animação em algum elemento. Para isso, dentro do seletor do elemento que queremos que sejá animado vamos utilizar algumas propriedades de animação chamada 'animation'.
São elas:

  • animation - pode ser utilizada para colocar todas as propriedades da animação.
  • animation-delay - especifica o tempo de espera até começar a animação, deve ser colocado em segundos.
    • animation-delay: 2s; - Inicia a animação apos 2 segundos.
  • animation-direction - especifica a direção que será executada a animação.
    • animation-direction: normal; - Valor padrão, a animação é feita para frente
    • animation-direction: reverse; - A animação é feita de tras para frente
    • animation-direction: alternate; - A animação vai para frente, depois volta
    • animation-direction: alternate-reverse; - A animação vai para tras, depois para frente
  • animation-duration - Especifica a duração da animacação, deve ser colocado em segundos.
    • animation-duration: 5s; - A animação toda vai acontecer em 5 segundos.
  • animation-interation-count - Informa a quantidade de vezes que a animação irá ocorrer.
    • animation-interation-count: 3; - Repete a animação 3 vezes.
    • animation-interation-count: infinite;
  • animation-name - Especifica o nome da @keyframe a ser utilizado
    • animation-name: myanimation - Irá executar a animação myanimation
