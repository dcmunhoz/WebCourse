Laços de repetição Javascript

Os laços de repetição são blocos de codigos que se repentem N vezes até que sua condição sejá verdadeira, ou até um valor especifico definido como escape.

Temos que tomar muito cuidado com as estruturas de repetições, pois se usadas de formas incorretas, ela pode entrar em um loop infinito, fazendo com que o programa trave ou fiquei muito lento.

No Javascript Temos 3 laços de repetições diferentes:

  • While
  • Do While
  • For
