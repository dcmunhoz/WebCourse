Estilo CSS (Cascading Style Sheets)

Para utilizar o css inline é necessário utilizar o atributo ' style="" ' na tag em que você desejá aplicar o estilo.
Este tipo de css não é muito utilizado, preaticament nunca utilizado, somente em alguns casos bem especificos e raros.

Vamos utilizar o css nos elementos abaixo, para isso é necessário, como dito, utilizar o atributo style. Após isso, vamos definir o seu valor, que pode ser um dos seguintes:

O primeiro valor, antes do ' : ' significa qual a edição que será feita, o valor apos os dois pontos, é justamente o valor dessa edição. Após o valor ser definido, é necessário finalizar a linha de edição utilizando ' ; '.

  • color: #FFF; - Color é utilizado para mudar a cor da fonte, seguido de um codigo Hexadecímal da cor Branca.
  • background-color: black; - Modifica a dor de fundo para a cor preta.
  • font-size: 20px; - Muda o tamanho da fonte para 20px;
