HTML5 - Compatibilidade

Com a adição de novas tags HTML5 vieram alguns problemas, como navegadores que não reconhecem as tags, por exemplo, as versões antigas do IE não irão entender nenhuma dessas tags.

Para resolvermos este problema, podemos utilizar o HTML5Shiv, este script foi totalmente feito em JS e faz com que nagevadores mais antifos do IE passem a reconhecer esta tag

Para isso, podemos ir no Google e digitar HTML5 Shiv e clicar no link do Git Hub

Logo em seguida, vamos até a seção de instalação e copiamos o codigo condicional para utilizar o html5shiv. Então colamos o mesmo dentro da nossa tag HEAD.

Agora temos que apontar para o script do html5shiv, para isso podemos utilizar 2 formas.
A primeira, via CDN e a segunda vamos fazer o download do script, colocar dentro de uma pasta e apostar para o mesmo.

Para utilizar o cdn, apos o SRC do comentario condicional, colocaremos o seguinte link: https://cdnjs.cloudflare.com/ajax/libs/html5shiv/3.7.3/html5shiv.min.js

Apos isso o html5shiv já esta pronto.
