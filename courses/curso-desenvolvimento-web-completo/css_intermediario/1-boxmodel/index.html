CSS Intermediario - Modelo de caixa

Todas as tags aprendidas até o momento tem algo em comum no css. Quando vamos formatar essas tags, elas se comportam como caixas.

Conteudo


Olhando o exemplo a cima, podemos ver como a div se comporta, parecendo como uma caixa.
Podemos notar varias coisas nas 2 divs de exemplo.
1 - A borda laranja esta em volta de toda a div, então tudo que esta dentro das bordas laranja é o conteudo da div.
2 - Podemos notar 2 espaçamentos diferentes nas divs, um dentro da caixa 1 e outro entre as 2 caixas. O espaçamento de dentro da primeira caixa é um espaçamento interno ao elemento
este espaçamento pode ser realizado utilizando o atributo 'padding'. Já o espaçamento entre as 2 caixas, é feito pela propriedade 'margin'. As edições de espaçamentos foram feitas na primeira caixa.
3 - Podemos ver que a tag que envolve o 'Conteudo' tambem funciona como uma caixa. Nessa caixa temos uma borda diferenciada para exemplificar melhor seu funcionamento.
