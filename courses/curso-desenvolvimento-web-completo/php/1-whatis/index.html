O que é PHP?

  • PHP é uma sigla para Hypertext Preprocessor . Esta é uma lingugem focada na programação do lado do servidor, mais conhecido como Back-end.

  • Seu uso é livre, basta fazer o download do mesmo e utilizar.

  • Outras caracteristicas do php: Compativel com diversos S.O., Paradigma estrutural ou Orientado a Objetos, Suporte a diversos bancos de dados.
