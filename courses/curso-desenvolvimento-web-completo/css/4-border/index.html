CSS - Bordas

Anteriormente já utilizamos bordas, porem vamos ter uma noção melhor de como elas funcionam.

Ao escolhermos em qual elemento vamos utilizar a borda, devemos colocar a seguinte sintaxe para o estilo alterado:
border: 1px solid red;

O primeiro valor, é a densidade da borda, quantos pixeis de expessura ela terá. Quanto maior o valor, mais grossa será a borda.

O segundo valor, é o formato da borda, no exemplo foi utilizado a borda SOLID, porem existem varios outros formatos, como nos exemplos abaixo:

Solid
Dotted
Dashed
Double
Groove

Esses são apenas alguns exemplos, neste link podemos ver muitos outros tipos de bordas.

E o terceiro valor, é a cor da borda.

É possivel também, escolher qual a cor, espessura e tipo de borda de cada lado, direita, esquerda, superior e inferior.
Para isso, devemos utilizar as propriedades especificas da borda.

Para cor: border-color;
Para espessura: border-width;
Para tipo: border-style;

Para entender melhor como definir os valores para especificos lados, vizualise a tabela:

Superior Direita Inferior Esquerda
border-color: red blue green yellow
border-width: 10px 5px 3px 8px
border-style: solid dotted dashed groove
