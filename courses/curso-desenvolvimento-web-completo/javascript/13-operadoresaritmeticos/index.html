Operadores Logicos Javascript

  • O que são?

    • Os operadores aritmeticos são todos os operadores da matematica.

      Exemplos:

Operador Nome Função
+ Adição Soma os valores
− Subtração Subtrai os valores
* Multiplicação Produto dos valores
/ Divisão Quociente dos valores
% Modulo Resto existente em uma operação de divisão
++ Incremento Pré/Pós Incremento
-- Decremento Pré/Pós Decremento
