Variaveis PHP

  • O que são Variaveis?
    • Variaveis na programação, são espaços temporarios que servem para o armazenamento de dados.
  • Tipos de variaveis
    • Int - tipo que guarda valores numéricos inteiros.
    • Float - tipo que guarda valores numéricos com casa decimal
    • String - Tipo que guarda textos
    • Boolean - tipo que guarda valores verdadeiros ou falsos
  • Regras para declaração de variaveis em PHP
    • Devem obrigatoriamente iniciar com o caracter "$"
    • Não podem ser iniciadas com "Espaços" ou números
    • Não é permitida a utilização de caracteres especiais (exeto o caracter "_")
  • Não requer (ou suporta) a definição explicita de tipo de variavel.
  • Case sensitive
