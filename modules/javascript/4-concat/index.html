Concatenar valores Javascript

O Javascript nos permite concatenar valores, o que isso significa?

Significa que podemos usar isso para deixar nosso site mais dinamico.
Para concatenarmos valores, devemos utilizar o caracter '+' entre os valores que queremos unir, ex:
var nome = 'Daniel';
var sobreNome = 'Munhoz'

console.log(nome + sobreNome);

O resultado desta operação será 'DanielMunhoz' tudo junto, pois não dizemos que existe um espaço entre as palavras, para resolver isso, basta concatenar com um espaço em branco da seguinte forma [var] + ' ' + [var2]
