CSS Intermediario - Posições

Trabalhando com posicionamento.

Existem varios valores para posicionamento, porem iremos trabalhar com os 3 principais e mais utilizados. Relative, Absolute,Fixed.
Para utilizar a propriedade de posição, no seletor desejado colocamos a pripriedade position, seguida de um dos três valores citados acima.

Posição - Relative

Ao utilizar a posição relativa, você consegue alinhar a div com base nela mesmo.

Ao utilizar o position relative, podemos utilizar 4 novas propriedades na nossa div, são elas: left, top, right e bottom.

Como exemplo, utilizamos "left:30px;" e "top:30px;" isso faz com que nossa div se mova 30px para a direita e 30px para baixo com base nela mesma.

Conclusão: Ao utilizarmos o valor relativa, conseguir MOVIMENTAR a div com base na sua posição inicial.

Neste segundo exemplo, utilizamos os valores -30px para left e top 0, e aumentamos sua largura para 620px. podemos ver que fizemos um efeito muito interessante, como se fosse um menu sobreposto na nossa pagina

Posição - Absolute

Ao utilizarmos o position absolute, podemos perceber que o nosso elemento quebra completamente o fluxo da pagina, não respeitando o elemento pai. Digamos que nós "removemos" o elemento de dentro da div pai, comportando-se como se não estivesse mais dentro dela. Com isso, o nosso elemento passa a ter relação de posição com base no elemento mais proximo que tenha uma posição.

Como antes do nosso elemento não temos nenhum outro com uma posição definida, a nossa div absoluta tomara como referencia o elemento body, pois já possui uma possition pré-definida.

Caso a div com id "conteudo-principa" possuisse uma position, nosso elemento ficaria absoluto baseado nessa div.

Ao utilizarmos position absolute, podemos tambem utilizar as propriedades top, left etc...

Vamos utilizar as propriedades bottom 0 e right. Com isso podemos perceber que nosso elemento desceu até o final da pagina posicionado a 30px da direita, pois como haviamos comentado, ele não respeita mais o fluxo dos outros elementos. ele passa a ser um elemento independente.

Utilizando a position absolute, podemos criar muitos efeitos interessantes, como por exemplo, uma barra lateral levando até as midias sociais.

Posição - Fixed

O position fixed pode ser parecido com o absolute, porem os 2 são diferentes. O fixed, fica referenciado ao nosos ViewPort. O que é isso? É toda a nossa area do monitor.

Sendo assim, o fixed não toma como base nenhuma tag, mas sim toda a area do nosso monitor.

O fixed é bastante utilizado quando queremos deixar algum elemento sempre em exibição na nossa dela, independente da rolagem da pagina. Um exemplo são os menus fixos no topo da pagina, que iremos replicar um agora.

No nosso elemento fixed, aplicamos as propriedades:

  • left:0;
  • top:0;
  • width:100%;
  • height: 50px;
  • background: rgba(30, 111, 68, .9); para conseguirmos um efeito de transparencia.
