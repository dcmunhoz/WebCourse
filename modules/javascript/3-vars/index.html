O que é Javascript

  • O que são variaveis?

    • Variaveis são espaçõs alocados temporariamento na memória que guardam um tipo de dado especifico e um valor.
      Quando definimos variaveis, seus dados estão salvos na memória, podemos futuramente recuperar o valor associado a variavel.

  • Tipos de variáveis
    • int - Variaveis do tipo numérico, guandam numeros interios, ex: 1, 52, 19, 10000.
    • float - Variáveis do tipo double são numeros que possuem casa decimal, ex: 4.55, 10.2, 14.425 .
    • string - Variáveis string armazenam qualquer tipo de texto, ex: 'Daniel costa Munhoz'
    • bool - Variáveis Boolean armazenam dados verdadeiros ou falso, ex: true.
  • Regras para declaração de variáveis
    • - Não podem ser inicializadas com numeros ou caracteres especiais, exeto underscore('_')
    • - Não podem ser utilizados caracteres especiais.
    • - Não podem ser iguais as palavras reservadas da linguagem.
  • Utiliza a palavra reservada "var" antes de declarar

    • Para declarar um variavel, utilizamos a palavra reservada 'var' seguida do nome e um ponto e virgula no final, ex: var _nomePaciente;

  • Case sensitive
    • Javascript é uma linguagem case sensitive, significa que da forma que você escreveu a variavel, aquela é a forma que será utilizada sempre.
      A linguagem diferencia letras minusculas de maiusculas
